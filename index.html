NAME

SYNOPSIS

[-h] {generate,deploy,config} ...

DESCRIPTION

A program to create and deploy compressed archives for software apps.

POSITIONAL ARGUMENTS

generate

Generate a compressed file for an app.

deploy

Deploy an app to a remote machine.

config

Configuration commands

COMMAND ’ generate’

usage: generate [-h] (--app-type APP_TYPE | --config FILE) [--overwrite] [--strict]

OPTIONS ’ generate’

--app-type APP_TYPE, -A APP_TYPE

The name of the app you want to generate.

--config FILE, -C FILE, --file FILE, -f FILE

Path to an app config file.

--overwrite, --over

Overwrite the existing app file.

--strict, -s

Raise errors instead of warnings.

COMMAND ’ deploy’

usage: deploy [-h] --host HOST --password PASSWORD --user USER --connection CONNECTION compressed-app-file
compressed-app-file

The path to the compressed file of the app you want to deploy.

OPTIONS ’ deploy’

--connection CONNECTION, -C CONNECTION

SSH Connection string (i.e. user:pass123@192.168.10.57).

COMMAND ’ config’

usage: config [-h] {set,get,reset} ...

POSITIONAL ARGUMENTS ’ config’

config set

Set the given setting with the given value.

config get

Get the value of the given setting.

config reset

Reset a single setting or all, to the default value.

COMMAND ’ config set’

usage: config set [-h] {generator.default_output_format,generator.output_path,generator.strict_by_default} value [value ...]
{generator.default_output_format,generator.output_path,generator.strict_by_default}

The name of the setting you want to change.

value

The value of the setting you want to set.

COMMAND ’ config get’

usage: config get [-h] {generator.default_output_format,generator.output_path,generator.strict_by_default}
{generator.default_output_format,generator.output_path,generator.strict_by_default}

The name of the setting you want to fetch.

COMMAND ’ config reset’

usage: config reset [-h] [--setting {generator.default_output_format,generator.output_path,generator.strict_by_default}]

OPTIONS ’ config reset’

--setting
{generator.default_output_format,generator.output_path,generator.strict_by_default}

The name of the setting you want to reset to default.
